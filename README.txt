Install node-js and npm
Change directory to 2 folder (client, server) and run command "node i --legacy-peer-deps"
Alfer done, open terminal, change directory to following folder
	server folder -> Run command 'npm run startServer' (server will run on PORT 5000)
				  -> Run command 'npm run startAuthServer' (server will run on PORT 5500)
	client folder -> Run command 'npm start' (client will run on PORT 3000)
